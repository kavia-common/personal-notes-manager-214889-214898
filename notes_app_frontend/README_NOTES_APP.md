# Ocean Notes - Kotlin Android (Compose)

This module contains a minimal, self-contained notes app using Jetpack Compose and an in-memory repository.

Features:
- Notes List with FAB
- Create/Edit screen (title required)
- View Note with Edit/Delete
- Ocean Professional theme colors
- Empty state and loading indicator

Build:
  ./gradlew :app:assembleDebug

Install/Run (device/emulator required):
  ./gradlew :app:installDebug
